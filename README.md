# 🚓 Police Data Analysis Python & Pandas

## 📌 Project Overview

This project focuses on analyzing a Police Traffic Stops dataset using **Python** and **Pandas** in **Jupyter Notebook**. The dataset contains information about traffic stops, including driver demographics, violation types, search records, and stop durations.

The objective of this project is to apply data cleaning, preprocessing, and exploratory data analysis (EDA) techniques to uncover meaningful insights from real-world law enforcement data. Through this analysis, various Pandas operations were used to manipulate, filter, and summarize the dataset efficiently.

---

## 📂 Dataset Information

The dataset consists of police traffic stop records with attributes such as:

* Driver Gender
* Driver Age
* Violation Type
* Search Conducted
* Stop Duration
* Speeding Details
* Traffic Stop Information

Each row represents an individual traffic stop observation.

## 📊 Analysis Tasks Performed

### 1. Data Cleaning

* Identified and removed columns containing only missing values.
* Improved dataset quality for accurate analysis.

### 2. Speeding Violation Analysis

* Determined whether male or female drivers were stopped more frequently for speeding violations.

### 3. Search Conducted Analysis

* Analyzed whether driver gender influenced the likelihood of a vehicle search during a stop.

### 4. Stop Duration Analysis

* Converted categorical stop durations into numerical values.
* Calculated the average duration of traffic stops.

### 5. Driver Age Analysis

* Compared age distributions across different violation categories.
* Generated statistical summaries using grouped analysis.
